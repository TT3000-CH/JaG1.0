# Jump and Groove V4

Mobile-first Portrait-Version fuer GitHub Pages.

Aenderungen V4:
- Startbild wird zuerst angezeigt und verschwindet per Tap.
- Danach startet das Spiel direkt.
- Kein Jump-Button im Spiel: Springen per Tap auf den Bildschirm.
- Kurzer Tap = kleiner Sprung, langer Tap = hoher Sprung.
- Niedrige und hohe Hindernisse.
- Punkte nur beim Einsammeln von Muenzen.
- Hindernisse laufen bis zum Kaenguru und kollidieren korrekt.
- Portrait-Layout fuer iPhone optimiert.

Deployment: Inhalt dieses Ordners ins GitHub-Repo hochladen und GitHub Pages auf main/root stellen.
